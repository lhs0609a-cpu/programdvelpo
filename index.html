FAQ

자주 묻는 질문

개발비와 월 관리비, 왜 따로인가요? +

개발비는 귀사 맞춤 프로그램을 만드는 일회성 비용이고, 월 관리비는 프로그램이 문제없이 계속 돌아가도록 유지하는 비용입니다. 건물을 지으면 관리비가 필요하듯, 프로그램도 외부 환경 변화(사이트 구조 변경, API 업데이트 등)에 대응하려면 지속적인 관리가 필요합니다.

월 관리 구독을 해지하면 프로그램은 어떻게 되나요? +

프로그램 소유권은 고객님께 있으므로 해지 후에도 그대로 사용 가능합니다. 다만, 해지 이후 발생하는 오류 수정이나 외부 변경 대응은 건당 별도 견적(시간당 10만원~)이 적용됩니다. 갑자기 프로그램이 멈췄을 때 긴급 수정비가 월 관리비보다 훨씬 비쌀 수 있어, 대부분의 고객이 관리 구독을 유지하십니다.

우리 회사 업무도 자동화가 가능할까요? +

엑셀로 하는 작업, 복붙이 많은 작업, 정해진 규칙이 있는 반복 작업이라면 거의 100% 자동화 가능합니다. 30분 무료 진단으로 귀사 업무의 자동화 가능 여부와 예상 효과를 정확히 알려드립니다.

개발 기간은 얼마나 걸리나요? +

간단한 자동화 도구는 1-2주, 중규모 시스템은 3-4주면 완성됩니다. 급하신 경우 긴급 개발도 가능합니다. 정확한 일정은 무료 상담 후 안내드리며, 약속한 납기는 100% 지킵니다.

필수는 아닙니다. 하지만 자동화 프로그램은 외부 사이트 변경, OS 업데이트, API 정책 변경 등으로 언제든 멈출 수 있습니다. 관리 없이 프로그램이 1주일 멈추면 직원 수작업 복귀 비용만 30만원 이상이고, 긴급 수정비는 별도입니다. Basic 15만원은 이런 리스크를 막아주는 보험이라고 생각하시면 됩니다.
